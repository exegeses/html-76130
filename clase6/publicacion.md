# Publicación de nuestro sitio

> Publicar nuestro sitio significa subirlo a la nube para que se pueda acceder desde cualquier equipo

> Tenemos varias alternativas para poder publicar el sitio.
> En este caso mencionaremos alternativas gratuitas y alternativas pagas

## Gratuitas

### Github Pages

> GitHub tiene una manera de publicar un sitio estático dentro de nuestro repositorio
> Para poder lograrlo tenemos que activar la opción Github Pages

1- Crear una cuenta de usuario en GitHub
2- En el avatar de nuestro perfil hacer clic en "Your repòsitories"
3- Utilizando el botón que dice "NEW" vamos a crear un nuevo repositorio

> Notas: En las opciones de configuración del nuevo repositorio la más importante es la que dice "Public", Ya que si elegimos Private nunca podremos publicar el repositorio
> El nombre del repositorio es opcional, si yo no lo asignó se va a asignar alguno aleatorio

4- Pulsar el botón que dice "Create Repository" en la parte inferior

> Con esta opción se crea el repositorio y ya estamos listos para subir nuestros archivos

5- Pulsar el botón que dice "Add file"
6- Usar el botón que dice "Upload files"
7- Hacer Arrastrar and Soltar con todos los archivos de nuestro proyecto
8- Ir a la parte inferior y hacer clic en el botón que dice "Commit Changes"

> En este punto ya subimos todos nuestros archivos.
> Es muy importante recordar que la página de inicio de nuestro sitio se llame "index.html"

9- En la parte superior pulsar el botón que dice "Settings"
10- En el lateral izquierdo ir a la opción que dice "Pages"
11- Ir la parte que dice "Branch" y en el desplegable que dice "None" seleccionar Main
12- Usar el botón "Save"

> Ya está todo listo
> Sólo hay que esperar uno o dos minutos a que nos muestre la URL del sitio
> Si no aparece, actualizar la página
